# Code for performing weighted backprojection

Installation
-----------------
To install, run 
```
    pip install -e .
```
in this directory.

Running the Code
-----------------
The weighted backprojection can be imported in your python files as 
```Python
    import wbpj.weighted_backprojection_1d
    volume = weighted_backprojection_1d(images, angles)
```
